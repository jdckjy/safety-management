# 수익 분석 보고서 (Research.md)

이 문서는 업로드된 Excel 파일에서 추출된 재무 데이터를 분석하고 시각화하는 방법에 대한 계획을 정의합니다. `ExcelParser.ts` 유틸리티를 통해 생성된 JSON 데이터를 기반으로 합니다.

## 1. 데이터 요약 (Summary)

가장 먼저, 전체 재무 상태를 한눈에 파악할 수 있는 핵심 지표들을 계산하고 표시합니다.

- **총 수입 (Total Income):** `type`이 `income`인 모든 `amount`의 합계
- **총 지출 (Total Expense):** `type`이 `expense`인 모든 `amount`의 합계
- **순수익 (Net Profit):** 총 수입 - 총 지출
- **거래 건수 (Total Transactions):** 전체 데이터 배열의 길이

## 2. 카테고리별 분석 (Category-based Analysis)

수입과 지출이 어떤 항목에서 발생했는지 세부적으로 파악하기 위해 카테고리별로 데이터를 집계합니다.

### 2.1. 지출 분석 (Expense Breakdown)

- 각 `category`별 지출액 합산
- 지출액이 가장 큰 상위 5개 카테고리 표시 (Top 5 Expense Categories)
- 결과를 시각화하기 위해 **원형 차트(Pie Chart)** 또는 **막대 차트(Bar Chart)** 사용을 권장합니다.

### 2.2. 수입 분석 (Income Breakdown)

- 각 `category`별 수입액 합산
- 수입액이 가장 큰 상위 5개 카테고리 표시 (Top 5 Income Categories)
- 마찬가지로 차트를 사용하여 시각적 표현을 강화합니다.

## 3. 시간 흐름에 따른 분석 (Time-series Analysis)

월별 또는 분기별 재무 추세를 파악하여 비즈니스 흐름을 이해합니다.

- **월별 수입/지출 추이:** `date` 필드를 기준으로 월별로 수입과 지출을 집계합니다.
- **라인 차트(Line Chart)**를 사용하여 시간의 흐름에 따른 수입과 지출의 변화를 시각화합니다. X축은 시간(월), Y축은 금액을 나타냅니다.

## 4. 주요 거래처 분석 (Key Client Analysis)

어떤 거래처와의 거래가 가장 활발한지 분석합니다.

- **상위 거래처 (Top Clients):** 거래액(`amount`)을 기준으로 상위 10개 거래처를 순위별로 나열합니다.
- 각 거래처별 총 거래액 및 거래 건수를 테이블 형태로 표시합니다.

## 5. UI/UX 구현 계획

위 분석 내용을 `ProfitAnalysis.tsx` 컴포넌트 내에 효과적으로 표시하기 위한 UI 계획입니다.

1.  **파일 업로드 영역:** 현재 구현된 파일 입력 UI를 유지합니다.
2.  **데이터 요약 대시보드:** 총 수입, 총 지출, 순수익을 표시하는 **Stat Card** 컴포넌트를 상단에 배치합니다.
3.  **분석 차트 섹션:**
    - 탭(Tabs)을 사용하여 '카테고리별 분석'과 '월별 추이'를 전환할 수 있도록 구성합니다.
    - '카테고리별 분석' 탭에는 지출 및 수입에 대한 파이 차트를 각각 표시합니다.
    - '월별 추이' 탭에는 월별 수입/지출 라인 차트를 표시합니다.
4.  **거래 내역 테이블:** 파싱된 전체 데이터를 표시하는 테이블을 하단에 배치하여 사용자가 원본 데이터를 확인할 수 있도록 합니다.

이 `Research.md` 문서를 바탕으로 `ProfitAnalysis.tsx` 컴포넌트를 고도화하고, 사용자에게 유의미한 재무 인사이트를 제공하는 화면을 개발할 것입니다.
