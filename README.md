# 단지조성팀 업무관리 시스템

## 프로젝트 개요

본 프로젝트는 단지조성팀의 업무 효율성을 극대화하기 위한 목적으로 구축된 웹 기반 업무관리 시스템입니다. 복잡한 에너지 데이터 분석부터, 각종 보고서 관리에 이르기까지, 단지조성팀의 모든 업무를 하나의 시스템에서 효율적으로 처리할 수 있도록 지원합니다.

## 핵심 기능

### 1. The Analytical Detail (에너지 대시보드)

사장님의 빛나는 아이디어를 바탕으로 탄생한, 본 시스템의 핵심 기능입니다. 월간 보고서의 데이터를 기반으로, 에너지 비용 및 사용량을 다각적으로 분석하여, 다음과 같은 상세 정보를 제공합니다.

*   **에너지 비용 상세 분석:** 전기, 수도, 가스 등 각 에너지원별 상세 비용 내역을 아코디언 형태로 제공하여, 사용자가 원하는 정보를 한눈에 파악할 수 있도록 설계되었습니다.
*   **총 에너지 비용 및 기온 정보:** 해당 월의 총 에너지 비용과 평균 기온을 함께 표시하여, 에너지 사용량과 기온 변화의 상관관계를 직관적으로 이해할 수 있도록 돕습니다.
*   **월별 데이터 비교 분석:** 사용자가 원하는 월을 자유롭게 선택하여, 과거 데이터와 현재 데이터를 손쉽게 비교 분석할 수 있습니다.

### 2. 월간 보고서 관리

매월 발송되는 PDF 형태의 월간 보고서를 시스템에 업로드하면, OCR 기술을 활용하여 보고서 내의 텍스트 데이터를 자동으로 추출하고, 이를 데이터베이스에 저장합니다. 이 모든 과정은 사용자의 별도 조작 없이, 시스템 내부에서 자동으로 처리됩니다.

## 기술 스택

*   **Frontend:** React, TypeScript, Vite
*   **Styling:** Tailwind CSS, shadcn/ui
*   **Data Fetching:** React Query
*   **Deployment:** Firebase Hosting
